# Public Folder

This folder contains promotional images for the bot.

## Required Images

Place the following image files in this folder:

1. **free_spin.jpg** - Image for "GET FREE SPIN ON ACE96AU" menu option
2. **hot_game_tips.jpg** - Image for "HOT GAME TIPS CHANNEL" menu option

## Supported Formats

- JPG/JPEG
- PNG
- Other image formats supported by Telegram

## Notes

- If images are not provided, the bot will send text-only messages
- Make sure image files are named exactly as specified above
